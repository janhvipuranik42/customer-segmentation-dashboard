# Customer Segmentation Dashboard

## Objective
Analyze customer purchasing behavior and segment customers based on sales performance and regional trends using an interactive dashboard.

## Tools Used
- Power BI
- Excel / CSV

## Features
- Customer Segment Analysis
- Region-wise Sales Comparison
- Customer Sales Distribution
- Category Filtering
- Customer Performance Analysis
- Interactive Dashboard Controls

## Dashboard Insights
- Premium customers contributed the highest sales.
- Sales performance varied across regions.
- Customer purchasing behavior differed between segments.
- A smaller group of customers generated a significant portion of total sales.
- Segmentation helps identify valuable customer groups.

## Dataset
Sample Superstore Dataset
